# 假新闻检测应用功能概述

## 核心功能

### 1. 内容检测

- 分析文本内容的真实性和可信度
- 识别潜在的虚假信息和误导性内容
- 提供详细的检测结果和分析报告

### 2. 多源内容提取

- **URL提取**：从网页链接中提取文本和图片内容
- **文件支持**：
  - PDF文档（支持文本提取和图片识别）
  - Excel表格（xlsx, xls）
  - Word文档（doc, docx）
  - 文本文件（txt）
  - Markdown文件（md）

### 3. 多媒体分析

- 支持图片内容分析
- 从PDF和Office文档中提取并分析图片
- 图片处理功能（复制、保存）

### 4. 历史记录管理

- 保存检测历史记录
- 查看和管理过往检测结果
- 支持历史记录导出和删除

### 5. 用户界面

- 深色/浅色主题切换
- 响应式设计，适配不同窗口大小
- 自定义右键菜单，提供快捷操作

### 6. 系统功能

- 浏览器数据清理
- 检测结果导出（HTML格式）
- 多窗口管理（主窗口、浏览器窗口）

### 7. 搜索与导航

- 内置搜索功能
- 链接快速访问
- 内容跳转能力

## 技术特点

- 模块化架构设计
- 集成大语言模型服务
- 支持多种文件格式和内容类型
